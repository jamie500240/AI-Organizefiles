# 📁 Script_GodCleanup (檔案清理與去重自動化) 使用說明書
## 大量檔案去重歸檔，含容量預估與模擬執行


## 1. 程式簡介

這是一支專業級的檔案整理工具，專門用來處理龐大、雜亂且可能含有大量重複檔案的資料夾。它會透過計算檔案的「真實內容 (SHA-256 雜湊值，即檔案的 DNA)」，精準識別並分離出「唯一檔案」與「重複檔案」。

**核心安全承諾：**

* **來源絕對唯讀：** 您的原始檔案絕對不會被修改、移動或刪除。程式只會「複製」檔案出來處理。
* **資料完整性驗證：** 每一份複製出來的檔案，程式都會重新核對一次 DNA，確認與原檔 100% 相同後才會進行歸檔，杜絕硬碟壞軌或傳輸造成的檔案損壞。

---

## 2. 操作步驟

**🛑 事前準備：**
如果您收到的是程式碼檔案（以 `.py` 結尾），請先確認您的電腦已安裝 Python 軟體。如果您收到的是已經打包好的應用程式（如 `.exe`），則不需安裝任何東西，直接往下看即可！

**步驟一：啟動程式**
點擊兩下執行程式（或在終端機執行腳本）。

**步驟二：選擇資料夾 (兩次彈窗)**
程式會依序跳出兩個視窗，請使用滑鼠選擇：

1. **來源資料夾 (Source)：** 您想要整理的「亂七八糟、充滿重複檔案」的資料夾。（程式對此資料夾唯讀）
2. **輸出資料夾 (Destination)：** 您想要存放「整理結果」的乾淨資料夾。（建議選在空間充足的磁碟或外接硬碟）

**步驟三：查看 DRY RUN (淺層掃描) 報告**
程式會先花幾秒鐘掃描來源資料夾，並在黑色視窗中印出一份報告。
請檢查：

* **總計容量** 與 **輸出地剩餘空間**（程式會自動加上 10% 的安全緩衝）。
* **預估耗時**（讓您心裡有底，是否需要放著讓它跑過夜）。

> 💡 *如果空間不足，程式會亮紅燈並自動停止。*

**步驟四：確認執行**
如果報告看起來沒問題，在視窗中輸入 `Y` 並按下 Enter 鍵，程式就會開始正式搬運。

> 💡 **溫馨提醒（重要）：**
> 若檔案量非常大（超過數十 GB），搬運可能需要好幾個小時。**請確保您的電腦已暫時關閉「自動睡眠/休眠」功能**，以免程式跑到一半被系統強制暫停。

## 3. 輸出結果與資料夾結構

任務完成後，請前往您的「輸出資料夾」，您會看到以下結構與一份 CSV 結算報表：

📂 **1_Unique_Archive (唯一歸檔區)**
這裡存放所有不重複的乾淨檔案。

* **特色：** 保留了原始的目錄結構，並且檔名乾淨無印記（適合直接拿來日常使用）。

📂 **2_Duplicates_Iso (重複檔案隔離區)**
這裡存放所有被判定為重複的檔案。

* **特色：** 這裡的檔名前面會帶有 DNA短碼 與 `_DUP` 印記。
* **用途：** 保留印記是為了方便您日後若需要使用其他工具進行「資料對帳」，可以直接看檔名核對。確認無誤後，您可以直接將整個資料夾刪除以釋放空間。

📂 **3_Failed_Items (異常處理區)**
如果在複製過程中遇到檔案被其他程式鎖死、權限不足、或是硬碟損壞導致無法讀取，程式會跳過它，並在這裡生成一個 `.txt` 紀錄檔，告訴您是哪個檔案出錯。

📄 **Process_Report.csv (結案報表)**
所有處理過的檔案去向，都會以表格形式記錄在這裡。包含「處理後名稱」、「DNA 短碼」、「狀態 (唯一/重複/失敗)」與「原始路徑」。

## 4. 進階自動化機制說明 (Phase 2)

為了讓您在整理完後能直接獲得最乾淨的環境，程式在最後階段會自動執行兩件事（您不需動手）：

**1. 印記清洗與防撞車機制 (自動改名)**

* 程式在搬運途中會加上 DNA 印記以防出錯，但在最後一刻，會把 `1_Unique_Archive` 裡面的印記全部洗掉，還原成乾淨的原始檔名。
* **防撞車：** 如果洗掉印記後發現同個資料夾有同名檔案，程式會自動補上「原始修改日期與編號」（例如：照片_20231025_01.jpg），確保檔案絕對不會被互相覆蓋。

**2. 目錄隧道扁平化 (自動拆除空房)**

* 為了避免整理完出現「點進去只有一個資料夾，再點進去只有一個檔案」的惱人深層隧道。
* 程式會自動從最底層檢查：只要該資料夾裡面少於 2 個檔案，程式就會把檔案「往上層移」，然後把空資料夾拆除。這會讓您的檔案結構變得非常清爽好找。

## 5. 狀況排除與自訂設定

* **我想中途停止怎麼辦？**
請直接在黑色視窗按下鍵盤的 `Ctrl + C`。
程式有斷點防護機制：它會立刻停止處理下一個檔案，銷毀正在複製到一半的殘缺檔案，並將「目前為止已經整理好的進度」寫入 CSV 報表。您可以安心中斷，不用擔心資料損壞。
* **如何開關特定功能？**
如果您用記事本（或程式碼編輯器）打開這個 `.py` 檔案，在最上方的 `[SET] 領域層` 中，您可以透過修改 `True` / `False` 來改變系統行為：
* `ENABLE_FILENAME_RESTORE = True` (改為 False 則唯一區的檔案將保留 DNA 印記)
* `ENABLE_DIR_FLATTENING = True` (改為 False 則不自動拆除單一檔案的資料夾)
